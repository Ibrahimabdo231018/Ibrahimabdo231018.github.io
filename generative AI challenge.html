The generative AI challenge

Stable Diffusion is a deep learning model capable of generating detailed images based on text descriptions. In this challenge we tried to do generate 2D game assets models using fine-tune Stable Diffusion model we choosed the item SWORD to generate and we had to follow some steps

  • 1. Study how diffusion models work.
  • 2. Select a fine-tuning method to use in this challenge.
  • 3. Create a dataset of game assets.
    • 4. Fine-tune a stable diffusion model with your dataset.
  • 5. Evaluate the results.

at first we understand how the model should work and then we saw a couple of methods to fine-tune the model we chosed Guide to Generate Infinite Game Assets by Fine-tuning Stable Diffusion, we created a dataset of game assets and then we fine-tuned the model with the dataset and then we evaluated the results. it was a wholesome chance to do all this
